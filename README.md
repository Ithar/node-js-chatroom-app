# node-js-chatroom-app
Chat room implmentation using socket.io

#### Features 
- Heroku hostest app 
- Sign-up 
- Different rooms
- Send location 
- Mustache templates

## Application Stack

Stack  | version |
--- | --- |

*test framework* | n/a

## NPM Modules (Local)
> npm install chalk

> npm install nodemon --save-dev

> npm install express

> npm install socket.io

## Application Run

cd node-js-chatroom-app

npm run dev

## Application URL

[[ Local ]]

http://localhost:3000
